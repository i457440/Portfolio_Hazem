Advanced Development

Voor de specialisatie advanced development heb ik eerst naar 10 libraries gezocht en daarvan 3 uitgewerkt. Dit heb ik gedaan met HTML, CSS, Javascript en Jquery. Met dit project kan ik ook mijn leeruitkomst development aantonen.

Vue.js

Ik heb 3 libraries uitgewerkt een van deze is projecten is Vue.js hiermee heb ik een fullpage scroll project gemaakt. Ik heb dit Met behulp van html css en javascript gemaakt. Als eerst ben ik naar vuejs.org gegaan en heb ik gekeken welke projecten er allemaal waren. Daarna heb ik ervoor gekozen om dit project te maken. Ik had geen grote problemen gehad tijdens het proces maar wel kleine probleempjes die zo waren opgelost.

Pixi.js

De 2e librarie die ik heb uitgewerkt is pixi.js. Als eerst ben ik naar pixijs.com gegaan en heb ik gekeken naar allemaal verschillende voorbeelden. Daarna heb ik pixi.js geïnstalleerd zodat ik het kon gebruiken. Vervolgens heb ik 1 voorbeeld gebruikt en heb ik daarmee gespeelt om te kijken hoe het werkt. Ten slotte was het niet moeilijk te gebruiken en was alles duidelijk.

Textillate.js

De 3e librarie die ik heb uitgewerkt is textillate.js. Als eerst ben ik naar textillate.js.org gegaan en ben ik naar de playground gegaan om verschillende animaties te zien en te testen. Daarna heb ik animate.css, lettering.js en jquery geïnstalleerd. Vervolgens heb ik gekeken van hoe ik de animaties die ik in de playground had gemaakt moest verwerken in de code, dit heb ik met behulp van een tutorial gedaan. Uiteindelijk is het gelukt en heb ik alleen problemen gehad met het linken van de librarie.
