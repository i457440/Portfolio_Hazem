Wireframe v1

Ik ben als eerst begonnen met het maken van een wireframe. Daarna heb ik feedback gevraagd.

1. In het startscherm moest ik het menu vermelden.

2. Ook moest ik onderscheid maken tusen afhaal, bezorg of restaurant.

3. De lijst in het keuze menu is aan de lange kant en daar moest ik gebruik maken van submenus.

4. Het laatste is het toevoegen van een back button zodat je altijd terug kan.

De feedback die ik heb gekregen is duidelijk en heb ik gelijk verwerkt!

Design v1

Ik ben begonnen met het maken van het design, deze heb ik aan de stijl gehouden van het restaurant. Het design heb ik gemaakt aan de hand van adobe xd. Daarna heb ik weer feeback gevraagd.

1. Om het verhaal af te maken was het leuk om een foto te zien waar je de qr code kunt scannen.

2. Bij de home pagina in plaats van fotos van het eten een foto van het restaurant laten zien zodat het er beter bij past, daarnaast moest ik de tekst verbeteren en moest ik de titel weghalen omdat het anders dubbel op is met het logo.

3. Bij de bestellen pagina de knop dranken boven in neerzetten inplaats van onderin.

4. En bij de gerechten pagina de lettertypen veranderen.

De docent was tevreden met het design alleen moest ik nog wat dingen finetunen. De gekregen feedback was duidelijk en heb ik direct verwerkt.

Wat heb ik ervan geleerd?

Ik heb geleerd hoe ik nu een gebruiksvriendelijkheid app kan maken met adobe xd. Ook kan ik dit project gebruiken voor mijn design en research leeruitkomsten omdat ik eerst research heb gedaan voordat ik ben gestart met het designen.

Hoe maakt mij dat een betere mediadesigner?

Door dit soort projecten te doen krijg je meer ervaring en leer je er ook veel van. Uiteindelijk kan ik dit voor 3 leeruitkomsten gebruiken. Ook vond ik het leuk om te doen omdat je zelf een onderwerp mocht kiezen en waarvoor je het wilt gebruiken.
