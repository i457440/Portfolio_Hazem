Welke code heb ik gebruikt

HTML
Om de basis te leggen heb ik eerst gewerkt met HTML. Met HTML heb ik 3 paginas gemaakt: de homepagina, de leeruitkomstenpagina en de projecten pagina. Doordat ik een template had gemaakt voor de leeruitkomstenpagina en de projectenpagina kon ik deze makkekijk dupliceren.

CSS
Om het naar mijn stijl te maken heb ik gebruik gemaakt van CSS. Dit heb ik gedaan met 3 CSS bestanden. Voor elke pagina heb ik een ander CSS bestand gebruikt om het gescheiden te houden zodat het niet met elkaar in conflict kan raken.

Javascript
In de website heb ik ook een paar javascript elementen toegevoegd. Onder andere heb ik gebruik gemaakt van AOS. Ook heb ik een active functie gemaakt voor het hoofdmenu en daarnaast heb ik gebruik gemaakt van de sticky navbar functie op de homepagina. Ten slotte heb ik textillate.js ingeschakeld om zo mijn advanced development aan te geven in mijn portfolio.

Feedback

Dit was alle feeback die ik heb gekregen tijdens mijn reviews.

1. Ze vonden het fijn dat ik een ontwerp had gemaakt en was begonnen met mijn realisatie.

2. Ik moest bij elke pagina nog een title element toevoegen. Daarbij moest ik elke pagina een duidelijke naam geven.

3. De javascript code moest ik in een aparte bestand zetten.

4. Ze vonden het fijn dat ik gebruik had gemaakt van github.

5. Ik moest bij mijn opdrachten aangeven wat voor feedback ik heb gekregen en wat ik ermee heb gedaan.

6. Ook moest ik eens kijken of ik een betere manier kon realiseren om vanuit een project terug te gaan naar de leeruitkomst waarin dit project staat. Omdat alles via de navigatie van het hoofmenu ging.

7. Ten slotte had ik ook een paar kleine bugs die ik moest oplossen.

De feedback was duidelijk en heb ik allemaal verwerkt.

Wat heb ik ervan geleerd?

Ik heb geleerd hoe ik veel HTML, CSS en javascript moet toepassen.
