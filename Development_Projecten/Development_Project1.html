Code

Hier heb ik versie 1 uitgebreid daarvoor heb ik meer gebruik gemaakt van javascript en css. Met het css heb ik de layout opgemaakt om het mooier en duidelijker te maken. Ook heb ik hier meer gebruik gemaakt van javascript. Vervolgens heb ik verschillende variablen en functions gemaakt om je zet op te slaan, geeft de computer hier een reactie op. Ten slotte komt er een uitslag uit en worden de scores weergegeven.

Wat heb ik ervan geleerd?

Hoe je met functions en variabelen een geautomatiseerde rock paper scissors game kunt maken. En hoe je gebruik moet maken van randoms, if en else if. Ten slotte heb ik ook ervaring opgedaan over hoe HTML en JavaScript samenwerken.

Hoe maakt mij dat een betere mediadesigner?

Door dit te doen krijg ik steeds meer ervaring binnen javascript. Ook moet je het leuk hebben als je dit soort opdrachten maakt en geeft het je een motivatie om door te gaan. Daarbij hoe meer projecten je maakt hoe meer ervaren je wordt.
