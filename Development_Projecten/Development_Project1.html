Rock paper sciccors

Github link:

Natuurlijk heb ik gebruik gemaakt van github om te laten zien wat voor stappen ik nou heb doorlopen. Als eerst heb ik poc exmaple 3 uitgewerkt wat op canvas werd aangegeven en daarna heb ik poc example 4 uitgewerkt.

Ga naar Github

Versie 1

Dit was versie 1 van mijn steen papier schaar game. Ik heb de stappen gevolgd die in de canvas course staan. Uiteindelijk heb ik geen problemen gehad en is het makkelijk gegaan.

Code

Versie 1 was niet veel code dus was het ook vlot gegaan. Wel was ik hier begonnen met het maken van een javascript poc. Vervolgens heb ik geleerd hoe ik gebruik moest maken van variablen en functies.

Versie 2

in POC example 4 heb ik POC example 3 uitgebreid. Aan de hand van icons, buttons, variablen en functions heb ik dat gedaan.

Code

Hier heb ik versie 1 uitgebreid daarvoor heb ik meer gebruik gemaakt van javascript en css. Met het css heb ik de layout opgemaakt om het mooier en duidelijker te maken. Ook heb ik hier meer gebruik gemaakt van javascript. Vervolgens heb ik verschillende variablen en functions gemaakt om je zet op te slaan, daarna slaat hij deze op en geeft de computer hier een reactie op en tenslotte komt er een uitslag uit en worden de scores aangegeven.

Wat heb ik ervan geleerd?

Hoe je met functions en variabelen een geautomatiseerde rock paper siccors game kunt maken. En hoe je gebruik moet maken van randoms, if en else if. Ten slotte heb ik ook ervaring opgedaan over hoe HTML en JavaScript samenwerken.

Hoe maakt mij dat een betere mediadesigner?

Door dit te doen krijg ik steeds meer ervaring binnen javascript. Ook moet je het leuk hebben als je dit soort opdrachten maakt en geeft het je een motivatie om door te gaan. Daarbij hoe meer projecten je maakt hoe meer ervaren je wordt.
