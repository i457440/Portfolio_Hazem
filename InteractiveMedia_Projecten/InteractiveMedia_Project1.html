Portfolio

Review 1

Dit was mijn eerste versie van de portfolio ik had daar nog niet heel veel maar dat is logisch als het je 4e week is. Daarop heb ik feedback gekregen tijdens mijn eerste review.

1. Ze vonden het fijn dat ik een ontwerp had gemaakt en was begonnen met mijn realisatie.

2. Ik moest bij elke pagina nog een title element toevoegen. Daarbij moest ik elke pagina een duidelijke naam geven.

3. De javascript code moest ik in een aparte bestand zetten.

4. En vonden ze het fijn dat ik gebruik had gemaakt van github.

De 1e review was prima gegaan, mijn beoordeling was geel.De gekregen feedback was duidelijk en heb ik direct verwerkt.

Review 2

Tijdens deze versie was ik begonnen met het vullen van de portfolio. Hier had ik 1 opdracht uitgewerkt. Uiteraard heb ik tijdens de 2e review ook feedback gekregen.

1. ik moest bij mijn opdrachten aangeven wat voor feedback ik heb gekregen en wat ik ermee heb gedaan.

2. Ook moest ik eens kijken of ik een betere manier kon realiseren om vanuit een project terug te gaan naar de leeruitkomst waarin dit project staat. Omdat alles via de navigatie van het hoofmenu ging.

3. Ten slotte had ik ook een paar kleine bugs die ik moest oplossen.

De 2e review was prima gegaan, mijn beoordeling was geel.De gekregen feedback was duidelijk en heb ik direct verwerkt.
